# Informe de la práctica: Despliegue de un sitio estático con GitHub Pages

Durante esta práctica he creado un pequeño sitio web estático utilizando HTML, CSS y JavaScript, y lo he gestionado mediante Git para llevar un control de versiones. He aprendido a inicializar un repositorio local, realizar commits con mensajes claros y conectar el proyecto con un repositorio remoto en GitHub.

Lo que más me ha costado ha sido la configuración inicial de GitHub, especialmente el uso del token de acceso personal para poder hacer `git push` por HTTPS, ya que no se utiliza la contraseña habitual de GitHub.

He aprendido la importancia de hacer commits atómicos, de documentar el proceso y de cómo GitHub Pages permite publicar automáticamente un sitio web a partir de un repositorio. Finalmente, he comprobado que cada cambio subido al repositorio se refleja en la web publicada.
